## Build

### Prerequisites

- NodeJS
- pnpm: `npm i -g pnpm`
- [Replugged](https://github.com/replugged-org/replugged#installation)

### Install

2. Clone this repository and cd into it
3. Install dependencies: `pnpm i`
4. Build the theme: `pnpm run build`
5. Reload Discord to load the theme
